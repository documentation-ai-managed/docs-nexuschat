## Recent Updates

<Update label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Launched voice message support with end-to-end encryption.
- Added file sharing up to `100MB` per message.

## Breaking Changes

- Updated authentication flow: Replace `api_key` with `access_token` in headers.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Old
fetch('https://api.example.com/messages', {
  headers: { 'api_key': 'YOUR_API_KEY' }
});

// New
fetch('https://api.example.com/messages', {
  headers: { 'Authorization': `Bearer ${YOUR_ACCESS_TOKEN}` }
});
```
```python
# Old
headers = {'api_key': 'YOUR_API_KEY'}

# New
headers = {'Authorization': f'Bearer {YOUR_ACCESS_TOKEN}'}
```
</CodeGroup>

## Overview

NexusChat empowers you to create, organize, and manage comprehensive documentation for your projects. You get a centralized space to build interactive docs with MDX support, version control, and seamless collaboration. Start organizing your knowledge base today with intuitive tools that scale with your team.

<Callout kind="info">
  NexusChat uses the brand color `{ #3B82F6 }` for primary elements. Apply it consistently in your custom themes: `background: #{3B82F6};` or `border-color: #{3B82F6};`.
</Callout>

## Quick Start

Get up and running with NexusChat in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://app.nexuschat.com` and create your free account.
  </Step>
  <Step title="Create Project" icon="plus">
    Click **New Project** and name it, like "My App Docs".
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Upload MDX files or use the editor. Here's a basic config example:
    
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      // nexuschat.config.js
      module.exports = {
        theme: {
          primaryColor: '#3B82F6'
        },
        pages: ['./docs/**/*.mdx']
      };
      ```
      ```python
      # nexuschat.py
      config = {
          "theme": {
              "primary_color": "#3B82F6"
          },
          "pages": ["./docs/**/*.mdx"]
      }
      ```
    </CodeGroup>
  </Step>
  <Step title="Publish" icon="rocket">
    Hit **Publish** to make your docs live at a custom URL.
  </Step>
</Steps>

## Brand Guidelines

Maintain NexusChat's visual identity.

- **Primary Color:** `{ #3B82F6 }` (use for CTAs, headers, links)
- **Typography:** Sans-serif fonts with bold headings
- **Icons:** Lucide library (e.g., `code`, `users`)

<Expandable title="Advanced Theming" default-open="false">
  Customize further with CSS variables:
  
  ```css
  :root {
    --nexus-primary: #3B82F6;
    --nexus-bg: #ffffff;
  }
  ```
</Expandable>
